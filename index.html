Features

Speed

Typical document processing is in time interval of under one second. RealIO is built on the Java 8 platform, which has the needed performance and tooling. RealIO engine uses efficiently implemented extraction strategies to boost performance using parallel recognition.

Ease of Integration

Web Services can be used to expose RealIO functionality to external or internal systems, as well as command line tool or RealIO tool library for direct batch processing. This allows straightforward integration with other systems.

Intelligent Classification

Classification is essential part of data extraction process to achieve better extraction results. RealIO can use machine learning techniques to classify documents based on recognized entities.

Configurability

Highly expressive domain specific API and scripting language is used to leverage the configurability of extraction and classification process. RealIO provides Document Viewer (DView) as a visual desktop application for better document visualization, extraction configuration and debugging.

Rich Representation

Textual and Visual document representation is used simultaneously to extract data, as each bit of information helps recognize important patterns.

Advanced Extraction

State-of-the-art recognition algorithms detect relevant data, enabling RealIO to outperform template based solutions.

RealIO Document Viewer

Document Viewer (DView) is a desktop application for:

  • Document visualization
  • Configuration of data extraction
  • Entity inspection

DView is a part of RealIO tools, used to facilitate RealIO in process of document data extraction. RealIO supports standard digital document formats, like PDF, DOC, TXT, XML, HOCR. RealIO can read the output of major OCRs, such as ABBYY and Tesseract.
