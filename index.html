Document Viewer (DView) is a desktop application for:

  • Document visualization
  • Configuration of data extraction
  • Entity inspection

DView is a part of RealIO tools, used to facilitate RealIO in process of document data extraction. RealIO supports standard digital document formats, like PDF, DOC, TXT, XML, HOCR. RealIO can read the output of major OCRs, such as ABBYY and Tesseract.
